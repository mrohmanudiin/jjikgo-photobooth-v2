# Jjikgo Photobooth - Deployment Guide

This guide covers deploying the backend to Railway and the frontends to Vercel. 
The system has been updated to support full Socket.io multi-branch sync and proper Database CRUD operations.

## 1. Deploy Backend to Railway

1. Push your entire `JJIKGO_PHOTOBOOTH` repository to GitHub.
2. Go to [Railway.app](https://railway.app), login, and click "New Project" -> "Deploy from GitHub repo".
3. Select your repository. **Make sure to set the Root Directory to `/backend`**.
4. Once created, add a **PostgreSQL** database service to your Railway project.
5. In your Backend service settings on Railway, add these **Variables**:
   - `DATABASE_URL` (Reference the PostgreSQL DATABASE_URL variable within Railway)
   - `JWT_SECRET`= (Any strong random string e.g., `your_super_secret_jwt_key`)
   - `PORT`=`3000` (Railway injects PORT dynamically, but defaults to 3000 if needed)
   - `FRONTEND_URL`=`*` (Or the explicit URLs of your Vercel domains later)
6. Railway will automatically run:
   - `npm install`
   - `npm run build` (Generates Drizzle client)
   - `npm run prestart` (Pushes Schema to DB)
   - `npm start` (Runs the Express server)
7. Grab the **Public Domain** generated by Railway for your Backend service.

## 2. Deploy Frontends to Vercel

You will deploy the 3 React apps (Admin, Staff, Cashier) to Vercel.

1. Go to [Vercel](https://vercel.com) and click "Add New" -> "Project".
2. Import your GitHub repository.
3. For **Admin Dashboard**:
   - Set Framework Preset to `Vite`.
   - Set **Root Directory** to `apps/admin-dashboard`.
   - Add Environment Variable: `VITE_API_URL` = `https://<YOUR-RAILWAY-BACKEND-URL>`
   - Click Deploy.
4. For **Cashier App**:
   - Set Framework Preset to `Vite`.
   - Set **Root Directory** to `apps/cashier-app`.
   - Add Environment Variable: `VITE_API_URL` = `https://<YOUR-RAILWAY-BACKEND-URL>`
   - Click Deploy.
5. For **Staff App**:
   - Set Framework Preset to `Vite`.
   - Set **Root Directory** to `apps/staff-app`.
   - Add Environment Variable: `VITE_API_URL` = `https://<YOUR-RAILWAY-BACKEND-URL>`
   - Click Deploy.
